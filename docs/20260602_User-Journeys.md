# User Journeys

Date: 2026-06-02

Purpose: map primary user journeys for residents, skeptics, supporters, officials, volunteers, and donors.

This is planning only. It is not final copy or UI.

## Core Journey

Strategic journey:

1. Understand what RCV is.
2. Understand how RCV works.
3. See why Melrose might consider it.
4. Understand the home rule petition pathway.
5. Trust the campaign because it is honest, sourced, and nonpartisan.
6. Join the email list, volunteer, donate, ask a question, or share the site.

Recommended route:

`/` -> `/what-is-rcv/` -> `/how-it-works/` -> `/why-melrose/` -> `/home-rule-petition/` -> `/faq/` -> `/contact/`

## Journey 1: Curious Resident

Starting mindset:

"I have heard of RCV, but I do not know how it works."

Entry points:

- Homepage from a shared link.
- Search result for "ranked choice voting Melrose."
- Social share.

Needs:

- Definition.
- Simple example.
- Local relevance.
- Low-pressure next step.

Recommended path:

`/` -> `/what-is-rcv/` -> `/how-it-works/` -> `/faq/` -> email signup.

Trust moments:

- Hero status note.
- Plain-language definition.
- Source links near claims.
- "Questions worth asking" section.

Conversion:

- Join updates.
- Share the explainer with a neighbor.

Failure risks:

- Too much legal detail too early.
- Overly promotional tone.
- No clear answer to "has Melrose adopted this?"

## Journey 2: Skeptical Resident

Starting mindset:

"This sounds confusing or risky. I want to know what they are not saying."

Entry points:

- `/myths-and-facts/`
- `/faq/`
- Concern-oriented homepage section.

Needs:

- Respectful tone.
- Valid concerns acknowledged.
- Clear distinction between myths and implementation questions.
- Official sources.

Recommended path:

`/myths-and-facts/` -> `/faq/` -> `/sources/` -> `/home-rule-petition/` -> `/contact/`

Trust moments:

- Acknowledgment that RCV does not guarantee cost savings, turnout, or campaign civility.
- Explanation of exhausted ballots.
- Legal-review placeholders.
- Official source links.

Conversion:

- Ask a question.
- Join updates to follow the process.
- Share source page with another resident.

Failure risks:

- Treating skepticism as bad faith.
- Calling all concerns myths.
- Hiding implementation questions.

## Journey 3: Civic-Minded Supporter

Starting mindset:

"I like the idea and want to know how to help."

Entry points:

- Homepage.
- `/why-melrose/`
- `/contact/`

Needs:

- Clear campaign goal.
- Concrete action.
- Confidence that the campaign is credible.

Recommended path:

`/` -> `/why-melrose/` -> `/home-rule-petition/` -> `/contact/`

Trust moments:

- Nonpartisan framing.
- Honest home rule path.
- Source log.
- Volunteer form with clear categories.

Conversion:

- Join email list.
- Volunteer.
- Donate once compliance path is ready.
- Share the site.

Failure risks:

- No immediate action path.
- No explanation of what volunteers do.
- Donation language not ready or legally unclear.

## Journey 4: Local Official Or Municipal Stakeholder

Starting mindset:

"Is this campaign serious, accurate, and respectful of process?"

Entry points:

- `/home-rule-petition/`
- `/melrose-elections/`
- `/sources/`
- `/about/`

Needs:

- Legal humility.
- Source reliability.
- Clear implementation questions.
- No invented local facts.

Recommended path:

`/home-rule-petition/` -> `/melrose-elections/` -> `/sources/` -> `/about/` -> `/contact/`

Trust moments:

- Article LXXXIX limitation described carefully.
- City Clerk/Elections Office role respected.
- Open questions listed plainly.
- Campaign site distinguished from official election resources.

Conversion:

- Contact the campaign.
- Request corrections.
- Share process concerns.

Failure risks:

- Implying official support.
- Overstating legal pathway.
- Using unreviewed ballot instructions.

## Journey 5: Volunteer Prospect

Starting mindset:

"I may help if I understand the work."

Entry points:

- Homepage join section.
- `/contact/`
- `/about/`

Needs:

- Role clarity.
- Low-friction signup.
- Ways to help beyond money.

Recommended path:

`/about/` -> `/contact/` -> volunteer interest.

Volunteer categories:

- Talk with neighbors.
- Help with events.
- Help with research/source review.
- Help with accessibility/plain-language review.
- Help with outreach.
- Help with donations/compliance after review.

Conversion:

- Volunteer interest form.
- Email list.

Failure risks:

- Asking for too much information.
- No follow-up expectation.
- No privacy note.

## Journey 6: Potential Donor

Starting mindset:

"I support the effort and may contribute."

Entry points:

- `/donate/`
- Homepage join section.
- `/about/`

Needs:

- Credibility.
- Clear legal/compliance status.
- Explanation of what support enables.
- Alternative ways to help.

Recommended path:

`/about/` -> `/donate/` -> donation flow or compliance placeholder.

Trust moments:

- Donation compliance language reviewed before launch.
- No invented entity status.
- Non-monetary options visible.

Conversion:

- Donate once verified.
- Join updates.
- Volunteer.

Failure risks:

- Premature donation form.
- Unclear recipient.
- Missing compliance disclaimers.

## Journey 7: Source Checker Or Journalist

Starting mindset:

"Can I verify this?"

Entry points:

- `/sources/`
- Footer.
- Citation links from any page.

Needs:

- Source log.
- Reliability labels.
- Access dates.
- Clear caveats.

Recommended path:

Any page -> `/sources/` -> `/contact/`

Conversion:

- Ask a source question.
- Submit correction.
- Cite the site cautiously.

Failure risks:

- Citation dumping without claim support.
- Advocacy sources not labeled.
- Stale source status.

## Journey Design Requirements

- Every journey should have an action that does not require agreement with RCV.
- Skeptical users should be able to ask questions without joining the campaign.
- Supporters should be able to take action after learning the pathway.
- Officials should see legal and administrative humility before calls to action.
- Donors should not see unreviewed compliance claims.
