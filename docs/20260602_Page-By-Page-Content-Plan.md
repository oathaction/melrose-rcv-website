# Page-By-Page Content Plan

Date: 2026-06-02

Purpose: define the content job, audience, CTAs, sections, claims, citations, placeholders, SEO, links, and UX notes for each recommended route.

This is a planning document. It is not final copy.

## Global Page Requirements

Every page should:

- Avoid claiming Melrose has adopted RCV.
- Avoid implying implementation is automatic.
- Link factual claims to sources or source-backed pages.
- Use nonpartisan language.
- Make legal and election-administration placeholders visible where relevant.
- Include a path to sources and contact/questions.
