# Project Brief

## Purpose

Create a nonpartisan civic education and organizing website for Melrose residents learning about ranked-choice voting and the possible Massachusetts home rule petition pathway.

## Current Phase

This document supports the initial scaffold. It does not contain final public-facing claims.

## Guardrails

- Do not state that Melrose has adopted ranked-choice voting.
- Do not state that implementation is automatic.
- Do not publish legal language before review.
- Keep placeholder claims clearly labeled.

## Near-Term Outcome

A working Astro scaffold with placeholder routes, accessible layout primitives, documentation, and citation infrastructure.
