Reliability key:

- High: official government, election administrator, statute, constitution, or primary legal/municipal source.
- Medium-high: nonpartisan civic/research organization, academic/research source, or professional association.
- Medium: advocacy source useful for framing and leads but requiring corroboration for factual/legal claims.
- Use with caution: media, local campaign, secondary source, or source with clear viewpoint; useful only when labeled.
