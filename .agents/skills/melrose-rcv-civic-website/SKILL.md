# Melrose RCV Civic Website

Use this skill when working inside the `melrose-rcv-website` repository.

## Mission

Build a beautiful, fast, accessible, nonpartisan civic website about ranked-choice voting for Melrose municipal elections through a Massachusetts home rule petition pathway.

## Guardrails

- Never say Melrose has adopted RCV unless that is verified in future source review.
- Never imply RCV implementation is automatic.
- Clearly distinguish verified facts, assumptions, open questions, and legal-review placeholders.
- Avoid invented citations, timelines, legal claims, and election-administration details.

## Content Style

- Write in plain language for residents.
- Be respectful of skeptical questions.
- Prefer source-grounded claims.
- Label placeholders visibly until reviewed.

## Engineering Style

- Use Astro, TypeScript, Tailwind CSS, and npm.
- Use semantic HTML and accessible components.
- Keep public routes working during each phase.
- Run `npm run build` before handing work back.
