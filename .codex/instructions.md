# Codex Instructions

Use this file as repo-local guidance for Melrose RCV work.

## Nonpartisan Civic Voice

- Keep language plain, civic, local, and respectful.
- Do not use partisan appeals.
- Assume readers include supporters, skeptics, and residents still learning.

## Legal And Factual Care

- Do not claim Melrose has adopted RCV.
- Do not claim implementation is automatic.
- Do not draft final petition language without legal review.
- Do not invent citations or legal conclusions.

## Build Standards

- Use Astro, TypeScript, Tailwind CSS, and npm.
- Keep placeholder content clearly labeled.
- Build reusable components and data modules.
- Run `npm run build` before handoff.
